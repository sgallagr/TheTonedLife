# thetonedlife.com

This repository contains the website of **The Toned Life**, a wellness business in Charlotte, NC.

The repository is maintained on behalf of the business owner. All business content — services, claims,
pricing, schedules, and offerings — is provided and approved by the business owner. The repository
maintainer provides technical services only and makes no representations regarding the business,
its services, or its products.

Built with [Jekyll](https://jekyllrb.com) and the Minimal Mistakes theme, hosted on GitHub Pages.
